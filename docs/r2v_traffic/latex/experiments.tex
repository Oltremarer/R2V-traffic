\paragraph{Experiments.}
We compare the baseline replay path against R2V-Traffic with diffusion proposals, the \texttt{not\_rare\_to\_val} story, and the full gate. The main study uses three seeds and reports average travel time, queue length, delay, throughput, and reward. We additionally run gate ablations (\texttt{no\_support}, \texttt{no\_ood}, \texttt{no\_dynamics}), story ablations, and sampling ablations. Candidate counts, gate pass rates, and artifact validation status are reported as integrity diagnostics, not as traffic performance metrics.
